# 微信小程序开发环境配置指南

本文档说明如何配置项目以支持域名 `dev.yys.lingyiqianfan.com` 访问和微信小程序开发。

## 一、SSL证书配置

### 1. 获取SSL证书

将SSL证书文件 `dev.yys.lingyiqianfan.com.jks` 放置到以下目录：
```
qianfanmall-all/src/main/resources/
```

### 2. 配置证书密码

在 `application-env-prod.yml` 中配置证书密码：
```yaml
qianfanmall:
  env:
    ssl:
      enabled: true
      key-store-password: your-keystore-password  # 替换为实际的SSL证书密码
      key-store: dev.yys.lingyiqianfan.com.jks
```

## 二、微信小程序配置

### 1. 配置小程序基本信息

在 `application-core.yml` 中配置微信小程序信息：

```yaml
qianfanmall:
  wx:
    mini-app:
      app-id: wxXXXXXXXXXXXX  # 替换为你的小程序AppID
      app-secret: xxxxxx      # 替换为你的小程序AppSecret
      mch-id: XXXXXXX         # 替换为商户号
      mch-key: xxxxxx         # API V2密钥
      api-v3-key: xxxxxx      # API V3密钥
      notify-url: https://dev.yys.lingyiqianfan.com/wx/pay/notify
```

### 2. 配置微信支付证书

将微信支付证书文件放置到以下目录：
```
qianfanmall-all/src/main/resources/wechat-certs/
```

需要以下文件：
- `apiclient_cert.p12` - 商户证书（PKCS12格式）
- `apiclient_key.pem` - 商户私钥
- `apiclient_cert.pem` - 商户证书

### 3. 微信公众平台配置

登录微信公众平台（mp.weixin.qq.com），进行以下配置：

#### 3.1 服务器域名配置
在「开发」->「开发管理」->「开发设置」->「服务器域名」中配置：

- **request合法域名**: `https://dev.yys.lingyiqianfan.com`
- **uploadFile合法域名**: `https://dev.yys.lingyiqianfan.com`
- **downloadFile合法域名**: `https://dev.yys.lingyiqianfan.com`

#### 3.2 业务域名配置
在「开发」->「开发管理」->「开发设置」->「业务域名」中配置：
- `https://dev.yys.lingyiqianfan.com`

## 三、微信公众号配置（可选）

如果需要使用公众号功能，在 `application-core.yml` 中配置：

```yaml
qianfanmall:
  wx:
    mp:
      app-id: wxXXXXXXXXXXXX  # 替换为你的公众号AppID
      app-secret: xxxxxx      # 替换为你的公众号AppSecret
      token: xxxxxx           # 消息推送Token
      aes-key: xxxxxx         # 消息推送AesKey
```

## 四、Docker部署配置

### 1. 证书目录挂载

将SSL证书和微信支付证书放到服务器的 `./certs` 目录：
```bash
mkdir -p ./certs
cp dev.yys.lingyiqianfan.com.jks ./certs/
cp -r wechat-certs ./certs/
```

### 2. 环境变量配置

创建 `.env` 文件或设置环境变量：
```bash
SSL_KEYSTORE_PASSWORD=your-keystore-password
MYSQL_PASSWORD=qianfanmall123456
```

### 3. 启动服务

```bash
docker-compose -f docker-compose.prod.yml up -d
```

## 五、微信开发者工具配置

### 1. 添加开发者

在微信公众平台的「成员管理」中添加开发者的微信账号。

### 2. 本地调试

在微信开发者工具中：
1. 设置 -> 代理设置 -> 不使用代理
2. 详情 -> 本地设置 -> 勾选「不校验合法域名」

## 六、配置文件说明

| 文件路径 | 说明 |
|---------|------|
| `application-env-prod.yml` | 生产环境配置（域名、SSL证书） |
| `application-env-dev.yml` | 开发环境配置 |
| `application-env.yml` | 通用环境配置（MySQL连接） |
| `application-core.yml` | 核心配置（微信、存储、通知等） |
| `application.yml` | 主配置文件 |

## 七、注意事项

1. **HTTPS要求**: 微信小程序要求所有网络请求必须使用HTTPS协议
2. **域名备案**: 服务器域名必须经过ICP备案
3. **证书有效期**: SSL证书有有效期限制，需要在过期前更新
4. **安全配置**: 不要将敏感信息（如AppSecret、商户密钥）提交到代码仓库

## 八、测试验证

### 1. 测试HTTPS访问

```bash
curl -k https://dev.yys.lingyiqianfan.com/admin/index/index
```

### 2. 测试微信登录

在小程序中调用 `wx.login()` 获取code，然后请求：
```
POST https://dev.yys.lingyiqianfan.com/wx/auth/login
Content-Type: application/json

{
  "code": "微信登录code"
}
```
