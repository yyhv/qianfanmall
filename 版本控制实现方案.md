# 一套代码实现三个版本的方案

## 概述

本文档详细说明如何通过**一套代码 + License控制**实现基础版、专业版、企业版三个版本的差异化功能。

**核心特点**：
- ✅ 同时支持管理后台和微信小程序
- ✅ License文件存储（不存数据库，防止篡改）
- ✅ RSA加密 + 数字签名（最高安全性）
- ✅ 后端统一验证（前端无法绕过）

---

## 核心思路

**关键原则**：一套代码 + License控制功能开关 = 三个版本

**实现方式**：
1. **所有功能都开发**（代码层面包含所有功能）
2. **通过License控制哪些功能可用**
3. **前端根据License动态显示/隐藏功能**
4. **后端API根据License进行权限拦截**

## 三、后端实现

### 3.1 License验证服务

```java
// LicenseService.java
@Service
public class LicenseService {

    private LicenseInfo licenseInfo;

    /**
     * 系统启动时加载License
     */
    @PostConstruct
    public void loadLicense() {
        File licenseFile = new File("license.lic");
        if (!licenseFile.exists()) {
            throw new RuntimeException("License文件不存在，系统无法启动");
        }

        // 1. 读取加密的License文件
        String encryptedLicense = FileUtil.readUtf8String(licenseFile);

        // 2. RSA解密
        String licenseJson = RSAUtil.decrypt(encryptedLicense, publicKey);

        // 3. 解析License
        JSONObject licenseObj = JSON.parseObject(licenseJson);
        JSONObject dataObj = licenseObj.getJSONObject("data");
        String signature = licenseObj.getString("signature");

        // 4. 验证数字签名（防止篡改）
        if (!RSAUtil.verify(dataObj.toJSONString(), signature, publicKey)) {
            throw new RuntimeException("License文件已被篡改，系统无法启动");
        }

        // 5. 解析License信息
        licenseInfo = dataObj.toJavaObject(LicenseInfo.class);

        // 6. 验证域名绑定
        String currentDomain = getCurrentDomain();
        if (!licenseInfo.getDomain().equals(currentDomain)) {
            throw new RuntimeException("License域名验证失败");
        }

        // 7. 验证是否过期
        if (licenseInfo.getExpireDate().before(new Date())) {
            throw new RuntimeException("License已过期，请联系供应商续费");
        }

        log.info("License验证成功：{} - {}", licenseInfo.getMerchantName(), licenseInfo.getEdition());
    }

    /**
     * 检查是否有某个模块的权限
     */
    public boolean hasModule(String moduleName) {
        return licenseInfo != null && licenseInfo.getModules().contains(moduleName);
    }

    /**
     * 获取License信息
     */
    public LicenseInfo getLicenseInfo() {
        return licenseInfo;
    }

    /**
     * 获取当前域名
     */
    private String getCurrentDomain() {
        try {
            InetAddress inetAddress = InetAddress.getLocalHost();
            return inetAddress.getHostName();
        } catch (Exception e) {
            return "localhost";
        }
    }
}
```

```java
@Aspect
@Component
public class ModulePermissionAspect {

    @Autowired
    private LicenseService licenseService;

    /**
     * 拦截带有@ModulePermission注解的方法
     */
    @Around("@within(modulePermission) || @annotation(modulePermission)")
    public Object checkModulePermission(ProceedingJoinPoint joinPoint,
                                       ModulePermission modulePermission) throws Throwable {

        String moduleName = modulePermission.value();

        // 检查License是否包含该模块
        if (!licenseService.hasModule(moduleName)) {
            throw new BusinessException("当前版本不支持此功能，请升级到更高版本");
        }

        return joinPoint.proceed();
    }
}
```

```java
// LicenseController.java
@RestController
public class LicenseController {

    @Autowired
    private LicenseService licenseService;

    /**
     * 管理后台获取License信息
     */
    @GetMapping("/admin/license/info")
    public Object getAdminLicenseInfo() {
        return getLicenseInfo();
    }

    /**
     * 微信小程序获取License信息
     */
    @GetMapping("/wx/license/info")
    public Object getWxLicenseInfo() {
        return getLicenseInfo();
    }

    /**
     * 公共方法：获取License信息
     */
    private Object getLicenseInfo() {
        LicenseInfo info = licenseService.getLicenseInfo();

        Map<String, Object> result = new HashMap<>();
        result.put("edition", info.getEdition());
        result.put("modules", info.getModules());
        result.put("expireDate", info.getExpireDate());
        result.put("merchantName", info.getMerchantName());

        return ResponseUtil.ok(result);
    }

    /**
     * 上传新License（用于版本升级）
     */
    @PostMapping("/admin/license/upload")
    public Object uploadLicense(@RequestParam("file") MultipartFile file) {
        try {
            // 1. 备份旧License
            File oldLicense = new File("license.lic");
            if (oldLicense.exists()) {
                File backupLicense = new File("license_" + System.currentTimeMillis() + ".bak");
                FileUtil.copy(oldLicense, backupLicense, true);
            }

            // 2. 保存新License文件
            File licenseFile = new File("license.lic");
            file.transferTo(licenseFile);

            // 3. 重新加载License
            licenseService.loadLicense();

            // 4. 返回新的License信息
            LicenseInfo newLicense = licenseService.getLicenseInfo();

            Map<String, Object> result = new HashMap<>();
            result.put("edition", newLicense.getEdition());
            result.put("modules", newLicense.getModules());
            result.put("expireDate", newLicense.getExpireDate());
            result.put("upgradeSuccess", true);

            return ResponseUtil.ok(result);

        } catch (Exception e) {
            return ResponseUtil.fail("License上传失败：" + e.getMessage());
        }
    }
}
```

```javascript
// 过滤路由（根据License）
export function filterAsyncRoutes(routes, modules) {
  const res = []

  routes.forEach(route => {
    const tmp = { ...route }

    // 如果路由需要特定模块，检查License
    if (tmp.meta && tmp.meta.module) {
      if (!modules.includes(tmp.meta.module)) {
        return  // 没有权限，跳过此路由
      }
    }

    // 递归处理子路由
    if (tmp.children && tmp.children.length > 0) {
      tmp.children = filterAsyncRoutes(tmp.children, modules)
    }

    res.push(tmp)
  })

  return res
}
```

```javascript
const app = getApp()

function request(options) {
  return new Promise((resolve, reject) => {
    wx.request({
      url: options.url,
      method: options.method || 'GET',
      data: options.data || {},
      success: (res) => {
        // 如果返回模块权限错误，提示用户升级
        if (res.data.errno === 500 && res.data.errmsg.includes('不支持此功能')) {
          wx.showModal({
            title: '功能不可用',
            content: '当前版本不支持此功能，请联系商家升级',
            showCancel: false
          })
          reject(res.data)
        } else {
          resolve(res.data)
        }
      },
      fail: (err) => {
        reject(err)
      }
    })
  })
}

export default request
```

```java
        licenseInfo.setExpireDate(request.getExpireDate());
        licenseInfo.setEdition(request.getEdition());
        licenseInfo.setMaxUsers(request.getMaxUsers());

        // 2. 根据版本设置模块
        List<String> modules = getModulesByEdition(request.getEdition());
        licenseInfo.setModules(modules);

        // 3. 构建License JSON
        JSONObject dataObj = (JSONObject) JSON.toJSON(licenseInfo);
        String dataStr = dataObj.toJSONString();

        // 4. 生成数字签名（使用私钥）
        String signature = RSAUtil.sign(dataStr, PRIVATE_KEY);

        // 5. 组装License
        JSONObject licenseObj = new JSONObject();
        licenseObj.put("data", dataObj);
        licenseObj.put("signature", signature);

        // 6. RSA加密整个License
        String licenseJson = licenseObj.toJSONString();
        String encryptedLicense = RSAUtil.encrypt(licenseJson, getPublicKey());

        return encryptedLicense;
    }

    /**
     * 根据版本获取模块列表
     */
    private List<String> getModulesByEdition(String edition) {
        List<String> modules = new ArrayList<>();

        // 基础版
        modules.add("BASIC_MALL");
        modules.add("WX_MINIPROGRAM");

        // 专业版
        if ("PROFESSIONAL".equals(edition) || "ENTERPRISE".equals(edition)) {
            modules.add("MARKETING_COUPON");
            modules.add("MARKETING_SECKILL");
            modules.add("MARKETING_GROUPON");
            modules.add("STATISTICS");
            modules.add("H5_MALL");
        }

        // 企业版
        if ("ENTERPRISE".equals(edition)) {
            modules.add("MEMBER_SYSTEM");
            modules.add("DISTRIBUTION");
            modules.add("POINTS_MALL");
            modules.add("MULTI_PLATFORM");
            modules.add("REDIS_CACHE");
        }

        return modules;
    }

    /**
     * 生成License ID
     */
    private String generateLicenseId() {
        SimpleDateFormat sdf = new SimpleDateFormat("yyyyMMdd");
        String dateStr = sdf.format(new Date());
        String randomStr = RandomStringUtils.randomNumeric(6);
        return "QF-" + dateStr + "-" + randomStr;
    }
}
```

## 八、安全机制说明

### 8.1 为什么前端控制不安全？

**小程序代码可以被反编译**：

1. **获取小程序包**：使用工具提取wxapkg文件
2. **反编译**：得到JavaScript源码
3. **修改代码**：删除权限检查逻辑
4. **重新打包**：生成新的小程序包

**但是！即使前端被破解，后端仍然安全**：

```java
// 攻击者修改小程序代码，删除了权限检查
// 但是当小程序调用API时，后端会拦截：

@GetMapping("/wx/seckill/list")
@ModulePermission("MARKETING_SECKILL")  // 后端强制验证
public Object list() {
    // 如果没有MARKETING_SECKILL权限
    // AOP拦截器会抛出异常：BusinessException("当前版本不支持此功能")
    return seckillService.list();
}
```

**结论**：
- ✅ **前端控制**：提升用户体验，避免显示无权限的功能
- ✅ **后端控制**：真正的安全保障，无法绕过

### 8.2 License文件防篡改机制

1. **数字签名**：使用RSA私钥对License数据签名
2. **RSA加密**：整个License文件使用RSA加密
3. **验证流程**：
   - 解密License文件
   - 提取数据和签名
   - 使用公钥验证签名
   - 如果数据被修改，签名验证失败

**商家无法修改License的原因**：
- ❌ 没有私钥，无法生成有效签名
- ❌ 修改数据后，签名验证失败
- ❌ 无法破解RSA加密

```java
    @Value("${license.file.path:license.lic}")
    private String licenseFilePath;

    @Value("${license.public.key:}")
    private String publicKey;

    /**
     * 系统启动时加载License
     */
    @PostConstruct
    public void loadLicense() {
        File licenseFile = new File(licenseFilePath);
        if (!licenseFile.exists()) {
            throw new LicenseException("License文件不存在，系统无法启动");
        }

        try {
            // 1. 读取加密的License文件
            String encryptedLicense = FileUtil.readUtf8String(licenseFile);

            // 2. RSA解密
            String licenseJson = RSAUtil.decrypt(encryptedLicense, publicKey);

            // 3. 解析License
            JSONObject licenseObj = JSON.parseObject(licenseJson);
            JSONObject dataObj = licenseObj.getJSONObject("data");
            String signature = licenseObj.getString("signature");

            // 4. 验证数字签名（防止篡改）
            if (!RSAUtil.verify(dataObj.toJSONString(), signature, publicKey)) {
                throw new LicenseException("License文件已被篡改，系统无法启动");
            }

            // 5. 解析License信息
            licenseInfo = dataObj.toJavaObject(LicenseInfo.class);

            // 6. 验证域名绑定
            String currentDomain = getCurrentDomain();
            if (!licenseInfo.getDomain().equals(currentDomain)) {
                throw new LicenseException("License域名验证失败");
            }

            // 7. 验证是否过期
            if (licenseInfo.getExpireDate().before(new Date())) {
                throw new LicenseException("License已过期，请联系供应商续费");
            }

            log.info("License验证成功：{} - {}", licenseInfo.getMerchantName(), licenseInfo.getEdition());

        } catch (Exception e) {
            log.error("License验证失败", e);
            throw new LicenseException("License验证失败：" + e.getMessage());
        }
    }

    /**
     * 检查是否有某个模块的权限
     */
    public boolean hasModule(String moduleName) {
        return licenseInfo != null && licenseInfo.getModules().contains(moduleName);
    }

    /**
     * 获取License信息
     */
    public LicenseInfo getLicenseInfo() {
        return licenseInfo;
    }

    /**
     * 获取当前域名
     */
    private String getCurrentDomain() {
        try {
            InetAddress inetAddress = InetAddress.getLocalHost();
            return inetAddress.getHostName();
        } catch (Exception e) {
            return "localhost";
        }
    }
}
```

#### 13.4.6 RSAUtil.java

```java
package com.lyqf.license.util;

import javax.crypto.Cipher;
import java.security.*;
import java.security.interfaces.RSAPrivateKey;
import java.security.interfaces.RSAPublicKey;
import java.security.spec.PKCS8EncodedKeySpec;
import java.security.spec.X509EncodedKeySpec;
import java.util.Base64;

/**
 * RSA工具类
 */
public class RSAUtil {

    /**
     * 生成RSA密钥对
     */
    public static void generateKeyPair() throws Exception {
        KeyPairGenerator keyPairGenerator = KeyPairGenerator.getInstance("RSA");
        keyPairGenerator.initialize(2048);
        KeyPair keyPair = keyPairGenerator.generateKeyPair();

        RSAPublicKey publicKey = (RSAPublicKey) keyPair.getPublic();
        RSAPrivateKey privateKey = (RSAPrivateKey) keyPair.getPrivate();

        String publicKeyStr = Base64.getEncoder().encodeToString(publicKey.getEncoded());
        String privateKeyStr = Base64.getEncoder().encodeToString(privateKey.getEncoded());

        System.out.println("公钥：" + publicKeyStr);
        System.out.println("私钥：" + privateKeyStr);
    }

    /**
     * 加密
     */
    public static String encrypt(String data, String publicKey) throws Exception {
        byte[] keyBytes = Base64.getDecoder().decode(publicKey);
        X509EncodedKeySpec keySpec = new X509EncodedKeySpec(keyBytes);
        KeyFactory keyFactory = KeyFactory.getInstance("RSA");
        PublicKey pubKey = keyFactory.generatePublic(keySpec);

        Cipher cipher = Cipher.getInstance("RSA");
        cipher.init(Cipher.ENCRYPT_MODE, pubKey);
        byte[] encryptedBytes = cipher.doFinal(data.getBytes());

        return Base64.getEncoder().encodeToString(encryptedBytes);
    }

    /**
     * 解密
     */
    public static String decrypt(String encryptedData, String privateKey) throws Exception {
        byte[] keyBytes = Base64.getDecoder().decode(privateKey);
        PKCS8EncodedKeySpec keySpec = new PKCS8EncodedKeySpec(keyBytes);
        KeyFactory keyFactory = KeyFactory.getInstance("RSA");
        PrivateKey privKey = keyFactory.generatePrivate(keySpec);

        Cipher cipher = Cipher.getInstance("RSA");
        cipher.init(Cipher.DECRYPT_MODE, privKey);
        byte[] decryptedBytes = cipher.doFinal(Base64.getDecoder().decode(encryptedData));

        return new String(decryptedBytes);
    }

    /**
     * 签名
     */
    public static String sign(String data, String privateKey) throws Exception {
        byte[] keyBytes = Base64.getDecoder().decode(privateKey);
        PKCS8EncodedKeySpec keySpec = new PKCS8EncodedKeySpec(keyBytes);
        KeyFactory keyFactory = KeyFactory.getInstance("RSA");
        PrivateKey privKey = keyFactory.generatePrivate(keySpec);

        Signature signature = Signature.getInstance("SHA256withRSA");
        signature.initSign(privKey);
        signature.update(data.getBytes());

        return Base64.getEncoder().encodeToString(signature.sign());
    }

    /**
     * 验证签名
     */
    public static boolean verify(String data, String sign, String publicKey) throws Exception {
        byte[] keyBytes = Base64.getDecoder().decode(publicKey);
        X509EncodedKeySpec keySpec = new X509EncodedKeySpec(keyBytes);
        KeyFactory keyFactory = KeyFactory.getInstance("RSA");
        PublicKey pubKey = keyFactory.generatePublic(keySpec);

        Signature signature = Signature.getInstance("SHA256withRSA");
        signature.initVerify(pubKey);
        signature.update(data.getBytes());

        return signature.verify(Base64.getDecoder().decode(sign));
    }
}
```

### 13.10 安全建议

#### 13.10.1 私钥保护

**不要将私钥放在JAR包中**，应该：

1. **使用配置文件**：
```yaml
license:
  public:
    key: ${LICENSE_PUBLIC_KEY}  # 从环境变量读取
```

2. **使用加密配置**：
```java
@Value("${license.public.key}")
private String publicKey;  // 从加密的配置文件读取
```

3. **私钥只保留在License生成工具中**：
   - License生成工具是内部工具，不对外发布
   - 私钥严格保密，不放入JAR包

#### 13.10.2 JAR包保护

1. **代码混淆**：使用ProGuard混淆
2. **数字签名**：对JAR包进行数字签名
3. **加密关键代码**：对RSA工具类进行特殊保护
